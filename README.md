# Student Management System

A full stack web application built using Java, Spring Boot, React, and MySQL to manage student records.

## Features
- Student registration
- Add / update / delete students
- REST APIs using Spring Boot
- React frontend dashboard
- Database integration with MySQL

## Tech Stack
Backend
- Java
- Spring Boot
- Hibernate
- JPA
- REST APIs

Frontend
- React.js
- HTML
- CSS
- JavaScript

Database
- MySQL
